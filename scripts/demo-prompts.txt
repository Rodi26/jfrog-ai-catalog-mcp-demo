JFrog AI Catalog MCP Demo — Copy-Paste Prompts & Commands
===========================================================

Primary demo surface is the JFrog AI Catalog UI.
These prompts are for the MCP Gateway portions (Act 4) and optional
MCP-admin-tools demo using the jfrog-mcp server.


=== ACT 4 — MCP GATEWAY SETUP (Terminal) ===

Step 1: Install the MCP Gateway (one-time, developer machine)

---BEGIN COMMAND---
export HOST_DOMAIN=yourcompany.jfrog.io
export PROJECT_KEY=ml-code-review
export CLIENT_ID=claude

bash <(curl -fL https://releases.jfrog.io/artifactory/jfrog-cli-plugins/mcp-gateway/latest/scripts/mcp-gateway.sh)
---END COMMAND---

Replace yourcompany.jfrog.io with your actual JFrog SaaS URL.
The script installs jf CLI + mcp-gateway plugin, authenticates the machine,
sets the active project context to ml-code-review, and configures Claude Code.

---

Step 2: Run the gateway (after Claude Code is configured)

---BEGIN COMMAND---
export PROJECT_KEY=ml-code-review
jf mcp-gateway run
---END COMMAND---

This starts the MCP Gateway. Claude Code connects to it and gets access
to the MCP servers approved for ml-code-review (with tool policies enforced).


=== ACT 4 — SHOW TOOL POLICY (optional CLI check) ===

Show which tools are accessible for the project (after gateway setup):

---BEGIN COMMAND---
jf mcp-gateway list-tools --project ml-code-review
---END COMMAND---

Expected output: filtered list showing only ^get_.* and ^list_.* tools,
with delete/* tools absent.

=== ACT 3 — DEVELOPER INTEGRATION (Code Snippet Reference) ===

Show this code snippet in Act 3 when demonstrating AI Gateway integration.
This is what JFrog generates in the "Use Model" pane.

---BEGIN CODE (Python)---
from openai import OpenAI

# JFrog AI Gateway — project-scoped token, not the provider's API key
client = OpenAI(
    api_key="<jfrog-project-scoped-token>",          # From: AI/ML > Registry > Use Model
    base_url="https://yourcompany.ml.jfrog.io/v1"   # JFrog AI Gateway endpoint
)

response = client.chat.completions.create(
    model="HuggingFace/facebook/bart-large-cnn",
    messages=[
        {"role": "user", "content": "Summarize this code function: def calculate_total(items): ..."}
    ]
)

print(response.choices[0].message.content)
---END CODE---

Key talking points for this code:
- api_key is a JFrog project-scoped token, not the provider's key
- base_url points to JFrog AI Gateway (ml.jfrog.io), not the provider
- model name format: "Provider/model-name"
- Standard OpenAI SDK — zero friction for developers switching providers


=== TROUBLESHOOTING PROMPTS ===

If MCP Gateway is not connecting, verify setup:

---BEGIN COMMAND---
jf mcp-gateway status
---END COMMAND---

If Claude Code doesn't see MCP tools, check config:

---BEGIN COMMAND---
cat ~/.claude.json | python3 -m json.tool | grep -A5 JFrogMCPGateway
---END COMMAND---

Verify project key is set:

---BEGIN COMMAND---
echo $PROJECT_KEY
---END COMMAND---

Should output: ml-code-review
